kirana Store Management System

This is a Kirana Store Management System developed using HTML, CSS and JavaScript. The system helps small grocery stores manage products, stock, billing, and customer records efficiently. It reduces manual calculation errors and improves store management workflow.

Technologies Used:

  • HTML
  • CSS
  • JavaScript

Project Features

  • Add / Update Customer Entry:
    Allows store owner to add new customer product details and update existing records.
    Reason: Helps maintain accurate and organized transaction records.

  • Customer Search System:
    Search functionality to quickly find customer records by name.
    Reason: Saves time when handling multiple customer entries.

  • Automatic Latest Transaction Display:
    Displays the most recently added or updated customer record.
    Reason: Helps track the latest activity instantly.

  • Structured Data Table View:
    All customer entries are displayed in a table format with date, price, and quantity.
    Reason: Makes data easy to read and manage.

  • Edit Record Option:
    Users can modify existing customer entries using the edit button.
    Reason: Allows correction of incorrect data without deleting records.

  • Delete Record Option:
    Provides functionality to remove unwanted customer records.
    Reason: Keeps the system clean and updated.

  • Date-wise Transaction Tracking:
    Each entry includes a transaction date.
    Reason: Enables tracking of daily and monthly sales.

  • Quantity With Unit Support:
    Supports quantities like kg, packet, litre, etc.
    Reason: Useful for grocery store inventory management.
