Snake Game

This is a classic Snake Game built using HTML, CSS and JavaScript. The objective of the game is to control the snake, collect food, and increase the score without hitting the walls.

Technologies Used:

  • HTML
  • CSS
  • JavaScript

Features:

  • 🎮 Smooth Snake Movement
  • 🍎 Food Collection System
  • 📈 Live Score Tracking
  • 📊 Show Previous Score History
  • 💀 Game Over Detection
  • 🔄 Restart Game Option
